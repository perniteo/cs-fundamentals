**1. 폭포수 모형의 특징**
   - 1979년 B.Boehm 개발 하향식 생명주기 모형
   - 단계별 정의가 명확히 구분
   - 단계별 산출물(문서, 코드, 도식)이 명확
   - 적용사례 많다
   - 요구사항의 변경이 어려움
   - 중복과정 발생
   - 초기 오류 → 엄청난 결과 발생

**2. 개발 단계** : 계획 - 요구사항 분석 - 설계 - 구현 - 시험 - 유지보수

**3. 폭포수 모형의 장단점**
   - 장점 : 적용 경험, 성공 사례가 많다. 단계별 정의 산출물이 정확하다.
   - 단점 : 과정 중 새로운 요구사항 반영이 어렵다.
