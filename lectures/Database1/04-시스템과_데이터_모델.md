1. 데이터베이스 시스템 구조와 데이터베이스에 접근하기 위하여 공통적으로 사용되는 함수들의 모임인 API가 있다.


2. 데이터베이스 시스템의 유형으로서는 중앙 집중식 데이터베이스 시스템과 여러 컴퓨터 시스템에서 분산되어 운영되는 분산 데이터베이스 시스템이 있다.


3. 데이터 모델이란 현실 세계를 데이터베이스에 표현하는데 사용되는 도구로서 데이터 구조와 연산, 그리고 제약조건으로 구성되어 있다.


4. 데이터 모델은 개념적 모델과 논리적 모델로 구분하고 논리적 데이터 모델에는 전형적으로 관계 데이터 모델, 계층 데이터 모델, 네트워크 데이터 모델이 있다.


5. 관계 데이터 모델의 릴레이션에서 사용되는 기본적인 용어로서 투플(Tuple), 속성(Attribute), 도메인(Domain) 등이 있다.
