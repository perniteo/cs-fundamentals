1. 관계 대수의 한계로 관계 대수 연산을 확장한 집단 함수, 그룹화, 세미 조인 등의 연산이 있다.
2. 집단 함수로 SUM, AVG, COUNT, MAX, MIN 이 있다.
3. 그룹화는 속성 값에 의해서 그룹으로 분할되고, 분할된 각 그룹에 대해 질의를 수행한다.
4. 관계 연산에서 관계 해석은 원하는 데이터만 명시하고 질의를 수행하는 방법은 명시하지 않는 선언적인 언어로서 투플 관계 해석과 도메인 관계 해석이 있다.
5. 투플 관계 해석은 원하는 릴레이션을 투플 해석식으로 표기하고 도메인 관계 해석은 릴레이션을 도메인 해석식으로 표현한다.
