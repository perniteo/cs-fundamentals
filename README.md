# cs-study

이 레포지토리는 **컴퓨터공학(CS) 기초 지식**과  
그 지식을 바탕으로 한 **설계 고민과 엔지니어링 사고 과정**을 함께 정리하기 위한 저장소입니다.

초기에는 전공 과목 정리를 목적으로 시작했지만,  
현재는 CS 지식을 실제 개발·설계 상황에 **어떻게 연결해 사고했는지**를 기록하는  
장기적인 엔지니어 학습 로그로 확장되었습니다.

---

## Repository Goals

- 컴퓨터공학 핵심 개념을 정리하고 복습한다
- 단순 암기가 아닌 **이해 중심의 정리**를 남긴다
- 실제 프로젝트 및 설계 과정에서의 **판단 근거와 사고 흐름**를 기록한다
- 엔지니어로 성장하는 과정을 장기적으로 축적한다

---

## What This Repository Is (and Is Not)

### ✔ This repository is:
- CS 기반 사고를 정리한 **엔지니어 학습 로그**
- 기술 선택의 **이유와 맥락**이 남아 있는 기록
- 장기적으로 계속 확장되는 개인 스터디 저장소

### ✘ This repository is not:
- 시험 대비용 요약 노트 모음
- 특정 기술 스택 튜토리얼
- 프로젝트 코드 저장소

---

## Philosophy

- **CS는 목적이 아니라 도구다**
- 중요한 것은 “무엇을 외웠는가”보다 **“왜 그렇게 설계했는가”**
- 설계는 한 번에 완성되지 않고 **진화한다**
- 고민한 흔적 자체가 실력이다

- 일부 문서는 특정 시점의 사고를 그대로 보존한다
