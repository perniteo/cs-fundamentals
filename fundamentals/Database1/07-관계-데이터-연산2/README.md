1. 조인 연산이란 두 릴레이션으로부터 연관된 투플들을 결합하는 연산으로서 세타 조인, 동등 조인, 자연 조인, 외부 조인, 세미 조인이 있다.
2. 세타 조인은 비교 연산자를 이용하여 다양한 조건을 표시하며, 두 릴레이션의 기본키와 외래키 관계에 해당하는 속성들이 세터 조인 조건을 만족하는 투플로 결과 릴레이션이 구성된다.
3. 세타 조인 중 동등 조인과 자연 조인이 있다.
4. 외부 조인은 상대 릴레이션에 대응되는 투플을 갖지 못하여도 결과 릴레이션의 투플로 남긴다. 왼쪽 외부 조인과 오른쪽 외부 조인, 완전 외부 조인이 있다.
5. 디비전은 R릴레이션의 속성이 S릴레이션의 속성 값을 모두 가진 R의 투플에서 S릴레이션이 가진 속성을 제거한 투플을 가지는 릴레이션이다.
