1. SQL(Structured Query Language)은 ANSI에서 데이터 언어의 표준으로 채택된 언어로서 다양한 상용 관계 DBMS에서 사용되고 있다.
2. SQL은 자연어에 가까운 구문을 사용하여 질의를 표현하는 비절차적인 언어이다.
3. SQL은 데이터 정의어와 데이터 제어어 및 데이터 조작어로 구성된다.
4. 데이터 정의어의 종류에는 CREATE, ALTER, DROP이 있다.
5. 테이블은 CREATE TABLE 구문으로 생성하며 이때 키와 각 속성의 제약 조건을 부여한다. 여기서 키는 기본키(PRIMARY KEY)와 외래키(FOREIGN KEY)를 식별하여 명세한다.
6. DROP TABLE 구문으로 테이블을 삭제하며 열의 추가 및 삭제를 ALTER TABLE 구문으로 수행한다.
