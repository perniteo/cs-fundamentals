1. 중첩 질의는 하나의 SQL문 처리 결과를 다른 SQL문에 전달하기 위하여 두 개 이상의 SQL문을 하나의 SQL문으로 연결한다.
2. 중첩 질의의 결과 하나의 스칼라 값을 반환하거나 하나의 속성에 대한 다중 값을 반환, 그리고 여러 속성으로 이루어진 릴레이션 반환한다.
3. 중첩 질의와 같이 사용하는 연산자는 IN, ANY, ALL, EXIST가 있다.
4. INSERT문은 기존의 릴레이션(테이블)에 투플을 삽입한다.
5. DELETE문은 한 릴레이션 내의 투플을 삭제한다.
6. UPDATE문은 한 릴레이션에 들어 있는 투플들의 속성 값을 수정한다.
