1. 프로젝트 관리 : 소프트웨어 프로젝트를 조직하고(organizing) 계획하고(planning) 일정 관리(scheduling)를 하는 것이다. 프로젝트 관리는 주어진 기간 내에 최소의 비용으로 사용자를 만족시키는 시스템을 개발하기 위한 전반적인 활용이다.
2. 매트릭스 (Metrics) :시스템, 컴포넌트 또는 프로세스가 주어진 속성을 어느 정도 만족하는가에 대한 수량적인 측정 크기 중심 매트릭스, 기능 중심 매트릭스
3. 프로젝트 관리(Project Management)의 구성 단계
   1. 프로젝트 계획 수립
   2. 프로젝트 가동
   3. 프로젝트 통제
   4. 프로젝트 종료
