1. 프로젝트 비용 산정 기법 :소프트웨어를 개발하는 데 소요되는 비용을 추정하는 것을 의미
2. 분류
   - 하향식 비용 산정 기법 : 프로젝트의 전체 비용을 산정한 후 각 작업별로 비용을 세분화 종류
   1. 전문가 감정 기법
   2. 델파이 기법
3. 상향식 비용 산정 기법 :프로젝트의 세부적인 작업 단위별로 비용을 산정한 후 집계하여 전체 비용을 산정하는 방법
4. 종류
   1. LOC(원시 코드 라인 수) 기법
   2. 개발 단계별 인월수 기법
   3. 수학적 산정 기법→COCOMO 모형, Putnam모형, 기능 점수(Function Point) 모형 
5. LOC 기법 :소프트웨어 각 기능의 원시 코드 라인 수의 비관치, 낙관치, 기대치를 측정하여 예측치를 구하고 이를 이용하여 비용을 산정한다.
6. LOC 기법 특징
   1. 측정이 용이하고 이해하기 쉬워 가장 많이 사용한다.
   2. 예측치를 이용하여 생산성, 노력, 개발 기간 등의 비용을 산정한다.
